1.深浅拷贝

浅拷贝：创建一个新对象，对象有着原始对象属性值的一份精确拷贝，如果属性是类型，拷贝的就是基本类型值，引用类型，拷贝的是内存地址，假设对象改变地址，则影响另一个对象

4.手写 instanceof
思路：a 沿着原型链找到 b.prototype,则 `a instanceof b = true`，遍历 a 的原型链，找到 b.prototype,返回 true，否则返回 false

```js
const my_instanceof = (A, B) => {
  let p = A //指针指向a
  while (p) {
    //遍历原型链
    if (p === B.prototype) {
      return true
    }
    p = p.__proto__
  }
  return false
}
```
